# EnTech Front-End

Front em reactjs e typeScript

## Rodar local 

Você vai precisar de [Node.js](http://nodejs.org/).

```sh
git clone git@github.com:lorenamelor/entech.git
npm i
npm start
```

O aplicativo vai rodar na url: [localhost:3000](http://localhost:3000/).
